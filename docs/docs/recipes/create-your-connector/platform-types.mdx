# Platform types

Social connectors' platforms can be divided into three types: Web, Native and Universal.

## Web

Social connectors with _Web_ platform type are build for desktop web applications.

## Native

_Native_ type social connectors are specially for those cases that users are required to go through authorization process in corresponding native mobile apps.

## Universal

_Universal_ type social connectors can work smoothly not only on desktop web applications, but also on mobile web applications via browser.
